How to use CO2SYS with IMP?

1. Install CO2SYS 
    Move to iMP/MATLAB directory and type 'git clone https://github.com/jamesorr/CO2SYS-MATLAB.git'.
2. Install a density calculator
    Type 'wget http://www3.mbari.org/Products/Matlab_shell_scripts/matlab/densatp.m'.
3. Choose 'co2sys' for co2chem
    Set 'def_co2sys = true' in caco3_main.m.
