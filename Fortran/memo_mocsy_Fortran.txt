Memo to use mocsy 

A. Install
    Follow the intruction at http://ocmip5.ipsl.jussieu.fr/mocsy/fort.html

B. Compile
    0) Switch on "mocsy" and "USE_PRECISION=2" in /input/defines.h
    1) Compile subroutine to call mocsy: 
        Type 'gfortran -ffree-line-length-none -fPIC -cpp -DUSE_PRECISION=2 -I/path/to/mocsy/directory -c call_mocsy.f90 -o call_mocsy.o'
    2) Compile subroutines for diagenesis & signal tracking:  
        Type 'gfortran -c -cpp -I/path/to/iMP/input caco3_test_mod_v5_6.f90'
    3) Compile 1 & 2: 
        Type 'gfortran -ffree-line-length-none -fPIC -cpp -I/path/to/iMP/input caco3_fortran.f90 caco3_test_mod_v5_6.o caco3_therm.o call_mocsy.o 
            -lopenblas -lmocsy -L/path/to/mocsy/directory'
        (*** Note: if you choose to use sparse matrix solver, add wapper and change reference library names, cf., readme_Fortran.txt) 

C. Run
    Type './a.exe ...'. See readme_Fortran.txt for the input (rr ...) you can specify when running the code.
